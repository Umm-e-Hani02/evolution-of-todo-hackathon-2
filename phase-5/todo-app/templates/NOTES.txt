=============================================================================
TROUBLESHOOTING
=============================================================================

If pods are not starting:

1. Check if Docker images are available in Minikube:
   eval $(minikube docker-env)
   docker images | grep phase4

2. If images are missing, load them into Minikube:
   minikube image load phase4-backend:latest
   minikube image load phase4-frontend:latest

3. Check pod events:
   kubectl describe pod -n {{ .Release.Namespace }} -l app.kubernetes.io/instance={{ .Release.Name }}

4. Verify secrets are created:
   kubectl get secret todo-app-secrets -n {{ .Release.Namespace }}
